# Unity-FPS-Multiplayer-Game
A FPS multiplayer game implemented using Unity3d.

Controls:

W - Move Forward <br />
S - Move Back <br />
A - Move Left <br />
D - Move Right <br />
TAB - Toggle cursor lock <br />
Left Mouse Click - Shoot <br />
C - Crouch <br />
R - Reload <br />
1/2/3 - Change Weapons <br />
SPACE - Jump <br />
